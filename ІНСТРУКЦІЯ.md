# Інструкція по роботі з сайтом Петриківський розпис

## Два способи додавати контент

### Спосіб A — через CMS (простіше)
1. Відкрийте ваш-сайт.netlify.app/admin/
2. Увійдіть з вашим email і паролем
3. Виберіть розділ (Майстри, Книги тощо)
4. Заповніть форму і натисніть Publish
5. Netlify автоматично оновить сайт за 2-3 хвилини

### Спосіб B — через VS Code
1. Створіть .md файл у потрібній папці
2. Додайте зображення у src/assets/
3. Виконайте команди:
   git add .
   git commit -m "опис змін"
   git push
4. Сайт оновиться автоматично

## Розміри зображень

| Тип | Розмір | Формат |
|---|---|---|
| Фото майстра (хедер) | 1200x800px | JPG |
| Фото майстра (картка) | 600x600px | JPG |
| Галерея робіт | 1200x1200px | JPG |
| Обкладинка книги | 400x600px | JPG |
| Фото осередку | 1200x600px | JPG |

Правила:
- Максимальний розмір файлу — 2 MB
- Формат — JPG або WebP
- Назви файлів — тільки латиниця без пробілів: tetyana-pata-portret.jpg
